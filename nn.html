Chapitre 4 : Équations différentielles du premier ordre

📖 Généralités Généralités

\( F(x,y,y') = 0 \) ou \( y' = f(x,y) \)

Définitions :

  • Une équation différentielle d'ordre 1 relie \( x, y, y' \).
  • Solution générale : famille de courbes dépendant d'une constante \( C \).
  • Solution particulière : déterminée par une condition initiale \( y(x_0)=y_0 \) (problème de Cauchy).

Classification : selon la forme (séparable, homogène, linéaire, Bernoulli, exacte, Riccati).

🔹 Variables séparables Séparables

\( g(y)dy = h(x)dx \)
  1. Mettre tous les \( y \) d'un côté, les \( x \) de l'autre.
  2. Intégrer : \( \int g(y)dy = \int h(x)dx + C \).
  3. Exprimer \( y \) si possible.
📘 Exemple détaillé : \( y' = 2x y \)
→ \( \frac{dy}{dx} = 2xy \) → \( \frac{dy}{y} = 2x dx \) (on suppose \( y\neq0 \))
→ \( \int \frac{dy}{y} = \int 2x dx \) → \( \ln|y| = x^2 + C \)
→ \( |y| = e^{x^2+C} = e^C e^{x^2} \) → \( y = K e^{x^2} \) avec \( K \in \mathbb{R} \).
La solution \( y=0 \) est incluse dans \( K=0 \).

🔹 Homogènes Homogènes

\( y' = f\left(\frac{y}{x}\right) \)
  1. Poser \( u = \frac{y}{x} \) ⇒ \( y = ux \) et \( y' = u + x u' \).
  2. Substituer : \( u + x u' = f(u) \) ⇒ \( x u' = f(u)-u \).
  3. Séparer : \( \frac{du}{f(u)-u} = \frac{dx}{x} \).
  4. Intégrer, puis remplacer \( u = y/x \).
🌟 Exemple détaillé : \( y' = \frac{y}{x} + \left(\frac{y}{x}\right)^2 \)
→ \( u = y/x \), alors \( u + x u' = u + u^2 \) ⇒ \( x u' = u^2 \) ⇒ \( \frac{du}{u^2} = \frac{dx}{x} \)
→ \( \int u^{-2} du = \int \frac{dx}{x} \) ⇒ \( -\frac{1}{u} = \ln|x| + C \) ⇒ \( u = -\frac{1}{\ln|x|+C} \)
→ \( y = ux = -\frac{x}{\ln|x|+C} \).

🔹 Linéaires Linéaires

\( y' + a(x)y = b(x) \)
  1. Facteur intégrant : \( \mu(x) = e^{\int a(x)dx} \).
  2. Multiplier : \( \frac{d}{dx}(\mu y) = \mu b(x) \).
  3. Intégrer : \( \mu y = \int \mu b(x) dx + C \).
  4. Exprimer \( y \).
📌 Exemple détaillé : \( y' + \frac{2}{x}y = x^2 \) (x>0)
→ \( \int a = \int \frac{2}{x} dx = 2\ln x \) ⇒ \( \mu = e^{2\ln x} = x^2 \)
→ \( (x^2 y)' = x^4 \) ⇒ \( x^2 y = \frac{x^5}{5} + C \) ⇒ \( y = \frac{x^3}{5} + \frac{C}{x^2} \).

🔹 Méthode de Bernoulli Bernoulli (linéaire)

\( y = u v \) avec \( u' + a(x)u = 0 \)
  1. Choisir \( u = e^{-\int a dx} \).
  2. Alors \( v' = \frac{b}{u} \) ⇒ intégrer \( v = \int \frac{b}{u} dx + C \).
  3. Solution : \( y = uv \).
✨ Exemple : même équation \( y' + \frac{2}{x}y = x^2 \)
→ \( u = e^{-2\ln x} = x^{-2} \), \( v' = \frac{x^2}{x^{-2}} = x^4 \) → \( v = x^5/5 + C \)
→ \( y = x^{-2}(x^5/5+C) = x^3/5 + C/x^2 \), identique.

🔹 Variation constante Lagrange

\( y = C(x) e^{-\int a dx} \)
  1. Solution homogène : \( y_h = C e^{-\int a dx} \).
  2. On pose \( C = C(x) \), on dérive et on substitue.
  3. On obtient \( C'(x) = b(x) e^{\int a dx} \) ⇒ \( C(x) = \int b e^{\int a dx} dx + K \).
  4. Donc \( y = e^{-\int a dx} \left( \int b e^{\int a dx} dx + K \right) \).
📙 Exemple : \( y' + 2y = e^{x} \)
→ \( \int a = 2x \), \( y_h = C e^{-2x} \), posons \( y = C(x)e^{-2x} \)
→ \( C'(x)e^{-2x} = e^{x} \) ⇒ \( C'(x) = e^{3x} \) ⇒ \( C(x) = \frac13 e^{3x} + K \)
→ \( y = e^{-2x}(\frac13 e^{3x}+K) = \frac13 e^{x} + K e^{-2x} \).

🔹 Bernoulli générale Bernoulli

\( y' + a(x)y = b(x) y^{n},\; n\neq0,1 \)
  1. Diviser par \( y^{n} \) : \( y^{-n}y' + a(x) y^{1-n} = b(x) \).
  2. Poser \( z = y^{1-n} \) ⇒ \( z' = (1-n) y^{-n} y' \).
  3. Obtenir : \( \frac{1}{1-n} z' + a z = b \) ⇒ \( z' + (1-n)a z = (1-n)b \).
  4. Résoudre cette équation linéaire en \( z \), puis revenir à \( y = z^{1/(1-n)} \).
🔧 Exemple détaillé : \( y' + \frac{1}{x}y = x y^2 \) (n=2)
→ Diviser par \( y^2 \) : \( y^{-2}y' + \frac{1}{x}y^{-1} = x \)
→ Poser \( z = y^{-1} \) ⇒ \( z' = -y^{-2}y' \) ⇒ \( -z' + \frac{1}{x}z = x \) ⇒ \( z' - \frac{1}{x}z = -x \)
→ Facteur intégrant \( \mu = e^{-\int 1/x dx} = 1/x \) ⇒ \( (z/x)' = -1 \) ⇒ \( z/x = -x + C \) ⇒ \( z = -x^2 + Cx \)
→ \( y = 1/z = \dfrac{1}{-x^2 + Cx} \).

🔹 Équation exacte Totale / Exacte

\( M(x,y)dx + N(x,y)dy = 0 \) avec \( \frac{\partial M}{\partial y} = \frac{\partial N}{\partial x} \)
  1. Vérifier la condition d'exactitude.
  2. Trouver \( F \) telle que \( \frac{\partial F}{\partial x} = M \) ⇒ \( F = \int M dx + \varphi(y) \).
  3. Dériver par rapport à \( y \) : \( \frac{\partial F}{\partial y} = \frac{\partial}{\partial y}\int M dx + \varphi'(y) = N \) ⇒ déterminer \( \varphi'(y) \).
  4. Intégrer \( \varphi'(y) \) pour obtenir \( \varphi(y) \).
  5. Solution implicite : \( F(x,y) = C \).
🎯 Exemple détaillé : \( (2xy+1)dx + (x^2+3y^2)dy = 0 \)
→ \( M_y = 2x \), \( N_x = 2x \) ⇒ exacte.
→ \( F = \int (2xy+1)dx = x^2y + x + \varphi(y) \).
→ \( F_y = x^2 + \varphi'(y) = x^2+3y^2 \) ⇒ \( \varphi'(y)=3y^2 \) ⇒ \( \varphi(y)=y^3 \).
→ Solution : \( x^2y + x + y^3 = C \).

🔹 Riccati Riccati

\( y' = a(x)y^2 + b(x)y + c(x) \)
  1. Connaître une solution particulière \( y_1 \) (souvent donnée ou devinée).
  2. Poser \( y = y_1 + \frac{1}{z} \).
  3. Substituer, simplifier → équation linéaire en \( z \) : \( z' + [b+2a y_1]z = -a \).
  4. Résoudre pour \( z \), puis \( y = y_1 + 1/z \).
💡 Exemple détaillé : \( y' = y^2 - \frac{2}{x^2} \) avec \( y_1 = \frac{1}{x} \)
→ Posons \( y = \frac{1}{x} + \frac{1}{z} \).
→ Calculer \( y' = -\frac{1}{x^2} - \frac{z'}{z^2} \). Remplacer :
\( -\frac{1}{x^2} - \frac{z'}{z^2} = \left(\frac{1}{x}+\frac{1}{z}\right)^2 - \frac{2}{x^2} \)
→ Développer : \( \frac{1}{x^2}+\frac{2}{xz}+\frac{1}{z^2} - \frac{2}{x^2} = -\frac{1}{x^2} + \frac{2}{xz} + \frac{1}{z^2} \)
→ Égalité : \( -\frac{1}{x^2} - \frac{z'}{z^2} = -\frac{1}{x^2} + \frac{2}{xz} + \frac{1}{z^2} \) ⇒ \( -\frac{z'}{z^2} = \frac{2}{xz} + \frac{1}{z^2} \)
→ Multiplier par \( -z^2 \) : \( z' = -\frac{2}{x}z - 1 \) ⇒ \( z' + \frac{2}{x}z = -1 \)
→ Facteur intégrant \( x^2 \) ⇒ \( (x^2 z)' = -x^2 \) ⇒ \( x^2 z = -\frac{x^3}{3} + C \) ⇒ \( z = -\frac{x}{3} + \frac{C}{x^2} \)
→ \( y = \frac{1}{x} + \frac{1}{-\frac{x}{3} + \frac{C}{x^2}} \).

🔍 Classification Choix de la méthode

Comment reconnaître et résoudre :

  • Séparable → séparer \( x \) et \( y \), intégrer.
  • Homogène → \( y' = f(y/x) \) → poser \( u=y/x \).
  • Linéaire → facteur intégrant ou Lagrange.
  • Bernoulli → transformation \( z = y^{1-n} \).
  • Exacte → vérifier \( M_y=N_x \), chercher \( F \).
  • Riccati → nécessite une solution particulière.
Chapitre 5 : Second ordre à coefficients constants

📌 Homogènes (coeffs constants) 2nd ordre homogène

\( a y'' + b y' + c y = 0 \)
  1. Équation caractéristique : \( a r^2 + b r + c = 0 \).
  2. Calculer \( \Delta = b^2 - 4ac \).
  3. Trois cas :
    • \( \Delta > 0 \) : racines \( r_1, r_2 \) → \( y = C_1 e^{r_1 x} + C_2 e^{r_2 x} \).
    • \( \Delta = 0 \) : racine double \( r \) → \( y = (C_1 + C_2 x) e^{r x} \).
    • \( \Delta < 0 \) : \( r = \alpha \pm i\beta \) → \( y = e^{\alpha x}(C_1 \cos(\beta x) + C_2 \sin(\beta x)) \).
🔥 Exemple 1 : \( y'' - 5y' + 6y = 0 \) → \( r^2-5r+6=0 \) ⇒ \( r=2,3 \) ⇒ \( y=C_1 e^{2x}+C_2 e^{3x} \).
🌸 Exemple 2 : \( y'' + 4y' + 4y = 0 \) → \( r^2+4r+4=0 \) ⇒ \( r=-2 \) double ⇒ \( y=(C_1+C_2 x)e^{-2x} \).
💜 Exemple 3 : \( y'' + 2y' + 5y = 0 \) → \( r^2+2r+5=0 \) ⇒ \( r=-1\pm 2i \) ⇒ \( y=e^{-x}(C_1\cos 2x + C_2\sin 2x) \).

📌 Non homogène (coeffs indéterminés) Méthode des coeffs indéterminés

\( a y'' + b y' + c y = f(x) \)
  1. Résoudre l'équation homogène → \( y_h \).
  2. Deviner \( y_p \) selon \( f(x) \) :
    • \( f(x)=P_m(x) \) → \( y_p = Q_m(x) \) (polynôme de même degré).
    • \( f(x)=k e^{kx} \) → \( y_p = A e^{kx} \).
    • \( f(x)= \cos(\omega x) \) ou \( \sin(\omega x) \) → \( y_p = A\cos(\omega x)+B\sin(\omega x) \).
    • Si un terme de \( y_p \) est solution de l'homogène, multiplier par \( x \) (ou \( x^2 \) si double racine).
  3. Substituer \( y_p \) dans l'équation, identifier les coefficients.
  4. Solution générale : \( y = y_h + y_p \).
📘 Exemple détaillé (pas de résonance) : \( y'' + 3y' + 2y = e^{2x} \)
→ \( y_h = C_1 e^{-x} + C_2 e^{-2x} \).
→ On pose \( y_p = A e^{2x} \) ⇒ \( y_p' = 2A e^{2x} \), \( y_p'' = 4A e^{2x} \)
→ Substituer : \( 4A e^{2x} + 6A e^{2x} + 2A e^{2x} = 12A e^{2x} = e^{2x} \) ⇒ \( A = \frac{1}{12} \)
→ \( y = C_1 e^{-x} + C_2 e^{-2x} + \frac{1}{12} e^{2x} \).
⚠️ Avec résonance : \( y'' - y = e^{x} \), \( y_h = C_1 e^{x} + C_2 e^{-x} \) → on pose \( y_p = A x e^{x} \) → après calcul, \( A = 1/2 \).

📌 Variation des constantes (Lagrange) 2nd ordre général

\[ y_p = -y_1 \int \frac{y_2 f(x)}{W} dx + y_2 \int \frac{y_1 f(x)}{W} dx \]
  1. Déterminer deux solutions indépendantes \( y_1, y_2 \) de l'équation homogène.
  2. Calculer le wronskien \( W = y_1 y_2' - y_2 y_1' \).
  3. Appliquer la formule pour obtenir une solution particulière \( y_p \).
  4. La solution générale : \( y = C_1 y_1 + C_2 y_2 + y_p \).
🎯 Exemple détaillé : \( y'' + y = \tan x \)
→ \( y_h = C_1 \cos x + C_2 \sin x \) ⇒ \( y_1=\cos x, y_2=\sin x \)
→ \( W = \cos x \cdot \cos x - \sin x (-\sin x) = \cos^2 x + \sin^2 x = 1 \)
→ \( y_p = -\cos x \int \frac{\sin x \cdot \tan x}{1} dx + \sin x \int \frac{\cos x \cdot \tan x}{1} dx \)
= \( -\cos x \int \frac{\sin^2 x}{\cos x} dx + \sin x \int \sin x dx \)
= \( -\cos x \int (\sec x - \cos x) dx + \sin x (-\cos x) \)
= \( -\cos x (\ln|\sec x + \tan x| - \sin x) - \sin x \cos x \)
= \( -\cos x \ln|\sec x + \tan x| + \cos x \sin x - \sin x \cos x \) ⇒ \( y_p = -\cos x \ln|\sec x + \tan x| \)
→ Solution générale : \( y = C_1 \cos x + C_2 \sin x - \cos x \ln|\sec x + \tan x| \).

📌 Synthèse des méthodes Résolutions pratiques

Stratégie pour les équations du second ordre à coefficients constants :

  • Homogène : toujours résoluble via l'équation caractéristique.
  • Non homogène :
    • Si \( f(x) \) est combinaison d'exponentielles, sinus/cosinus ou polynômes → méthode des coefficients indéterminés (rapide).
    • Si \( f(x) \) est quelconque (tangente, logarithme, sécante, etc.) → variation des constantes (plus générale).
  • Ne pas oublier le principe de superposition : \( y = y_h + y_p \).
📌 Exemple complet : \( y'' + 4y = 5\cos(2x) \)
→ \( y_h = C_1 \cos 2x + C_2 \sin 2x \).
→ \( \cos 2x \) est solution de l'homogène → on essaie \( y_p = A x \cos 2x + B x \sin 2x \).
→ On calcule \( y_p', y_p'' \), on substitue : après simplification on obtient \( A=0, B=5/4 \)
→ \( y = C_1 \cos 2x + C_2 \sin 2x + \frac{5}{4}x \sin 2x \).
